# GitHub Portfolio - Modern, Responsive Website

A production-ready, fully responsive GitHub portfolio website built with **pure HTML5, CSS3, and Vanilla JavaScript (ES6+)**. No frameworks, no dependencies—just clean, semantic code.

## ✨ Features

- ✅ **Fully Responsive** - Mobile, Tablet, Desktop optimized
- ✅ **Dark/Light Mode** - Toggle with localStorage persistence
- ✅ **Smooth Navigation** - Sidebar menu with smooth transitions
- ✅ **Form Validation** - Real-time client-side validation
- ✅ **Semantic HTML5** - Proper SEO and accessibility
- ✅ **Modern CSS** - Flexbox/Grid with animations
- ✅ **Performance Optimized** - Minimal code, fast load times
- ✅ **No Dependencies** - Pure vanilla JavaScript ES6+
- ✅ **Production Ready** - No build step required
- ✅ **Mobile Hamburger Menu** - Auto-hide on mobile devices
